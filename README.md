# Gallery App - Unsplash Photo Gallery

React приложение для просмотра фотографий из Unsplash с возможностью авторизации и лайков.

## Технологии

- React 18
- Redux Toolkit + Redux Saga
- React Router v6
- Axios
- Unsplash API
- CSS Modules

## Возможности

✅ Лента фотографий с бесконечной прокруткой  
✅ Детальный просмотр фотографии  
✅ OAuth авторизация через Unsplash  
✅ Лайки фотографий  
✅ Responsive дизайн
